## Plan Your Day

Use the **Timer** to work in focused sessions with automatic time logging.

### Plan Day (Ctrl+Y Ctrl+T)

1. Press `Ctrl+Y Ctrl+T` (or `Cmd+Y Cmd+T` on Mac)
2. Choose number of work units
3. Assign issues/activities to each unit
4. Start timer — 45min work + 15min break cycles

### Or Stage in Kanban First

1. Open **Kanban** view in sidebar
2. Add tasks with notes and priorities
3. Click **Add to Today's Plan** when ready

### Quick Log (Ctrl+Y Ctrl+Y)

Skip the timer? Log time directly:
1. Press `Ctrl+Y Ctrl+Y`
2. Select issue, enter hours (e.g., `2h`, `30m`, `1.5`)
3. Done — entry appears in Time Entries
